Widget for displaying BuzzCity Ads (for mobile sites)

== Description ==

http://www.buzzcity.com/

BuzzCity is a global advertising network. Earn advertising revenue by serving network ads on your mobile sites and apps, and get paid for every click that you receive. Our global relationships with advertisers ensure that you earn optimal revenue from key mobile markets worldwide.

Note: You will need a BuzzCity Partner ID. Register as a Publisher at http://www.buzzcity.com

== Installation ==

1. Install the BuzzCity Ads Widget plugin to your blog.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the Widget on your blog through the Appearance->Widgets menu
4. Set up the Widget by entering your BuzzCity Partner ID, and selecting the type of ad banner.
5. Done!
